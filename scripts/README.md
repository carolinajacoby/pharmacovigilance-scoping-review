# Data Analysis – Scoping Review

This repository contains the script used for data analysis in the scoping review presented in this study. The code was developed to organize, process, and analyze the information extracted from the included studies, enabling the reproduction of the analytical steps described in the manuscript.

The workflow covers data preprocessing as well as descriptive analyses and, when applicable, additional approaches such as clustering and pattern exploration. The main goal is to ensure transparency, reproducibility, and traceability of the results.

The data used follow the structure defined during the data extraction phase of the review, and the script can be adapted for similar applications in other scoping review studies.

## Structure

- `scope_review_data_analysis.ipynb`: main notebook containing all data analysis steps

## Requirements

The analyses were developed in a Python environment. The following libraries are recommended:

- pandas  
- numpy  
- matplotlib  
- seaborn  
- scikit-learn  

## Reproducibility

To reproduce the results:

1. Ensure all required libraries are installed  
2. Run the notebook `scope_review_data_analysis.ipynb` sequentially  
3. Use the data structured according to the review extraction process  

## Notes

This script can be adapted for other studies involving scoping reviews, especially those using structured data extraction and analytical workflows.
